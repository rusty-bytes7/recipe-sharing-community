# Recipe Sharing Community

Welcome to the **Recipe Sharing Community**! This application allows users to share, browse, and discover delicious recipes contributed by the community.

## 🚀 Features

- Share your favorite recipes with the world
- Browse and search recipes by ingredients or categories
- Responsive and user-friendly interface

---

## 🖥️ Running Locally

Follow these steps to set up the project on your local machine.

### 1. Clone the Repository

```bash
git clone https://github.com/rusty-bytes7/recipe-sharing-community.git
cd recipe-sharing-community
```

### 2. Install Dependencies

This project uses npm (Node.js package manager). If you don’t have Node.js and npm installed, [download them here](https://nodejs.org/).

```bash
npm install
```

### 3. Start the Local Server

```bash
npm start
```

This will start the development server. Open your browser and visit:

```
http://localhost:3000
```

The app should now be running locally!

## 💡 Troubleshooting

- If you run into issues with missing packages, ensure you’ve run `npm install`.
- If the app doesn’t start, check for errors in the terminal and verify your Node.js version (preferably v14 or higher).
